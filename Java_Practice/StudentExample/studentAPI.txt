Student -- a class to manage students registered in a course (see Student.java)


List of constructors:
Student(String)
constructor -- create Student object with student name


Student(String, double[]) 
constructor -- create Student object with name and list of grades



List of instance methods:
void setName(String)
method -- update name of the Student

void updateCourseGrades(double[])
method -- update point grades for the Student (without modifiying or deleting existing grades)

String toString()
method -- return a summary for this Student that includes only the student ID and whether the course was completed

String listGrades()
method -- return the student's name and list of grades earned


List of static methods
static int getNumStudents()
method -- returns the number of Student objects
